====== КОМАНДЫ ДЛЯ УСТАНОВКИ NGINX КОНФИГУРАЦИИ ======

ПАПКИ НА СЕРВЕРЕ:
- Маркетинг: /var/www/prohelper_marketing/client
- Личный кабинет: /var/www/prohelper_lk

=== ВАРИАНТ 1: Через SCP (с локальной машины) ===

1. Загрузить конфигурации на сервер:
scp deploy/nginx/1мост.рф.conf root@ваш-сервер:/tmp/
scp deploy/nginx/lk.1мост.рф.conf root@ваш-сервер:/tmp/

2. Подключиться к серверу:
ssh root@ваш-сервер

3. Установить конфигурации:
sudo cp /tmp/1мост.рф.conf /etc/nginx/sites-available/
sudo cp /tmp/lk.1мост.рф.conf /etc/nginx/sites-available/
sudo ln -sf /etc/nginx/sites-available/1мост.рф.conf /etc/nginx/sites-enabled/
sudo ln -sf /etc/nginx/sites-available/lk.1мост.рф.conf /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default

4. Проверить и применить:
sudo nginx -t
sudo systemctl reload nginx

=== ВАРИАНТ 2: Прямо на сервере ===

1. Подключиться к серверу:
ssh root@ваш-сервер

2. Создать файлы конфигурации:
sudo nano /etc/nginx/sites-available/1мост.рф.conf
(скопировать содержимое из deploy/nginx/1мост.рф.conf)

sudo nano /etc/nginx/sites-available/lk.1мост.рф.conf
(скопировать содержимое из deploy/nginx/lk.1мост.рф.conf)

3. Активировать конфигурации:
sudo ln -sf /etc/nginx/sites-available/1мост.рф.conf /etc/nginx/sites-enabled/
sudo ln -sf /etc/nginx/sites-available/lk.1мост.рф.conf /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default

4. Проверить и применить:
sudo nginx -t
sudo systemctl reload nginx

=== ПРОВЕРКА РАБОТЫ ===

Проверить редиректы:
curl -I https://1мост.рф/dashboard
curl -I https://1мост.рф/login

Ожидаемый результат: 
HTTP/1.1 301 Moved Permanently
Location: https://lk.1мост.рф/...
