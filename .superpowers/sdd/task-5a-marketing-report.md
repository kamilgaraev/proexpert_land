# Task 5A — отчёт по публичной коммерческой модели МОСТ

## Результат

Публичные маркетинговые поверхности переведены на единую модель: бесплатная ограниченная база, десять независимых бизнес-пакетов, полный комплект и корпоративные условия.

- `/pricing` заменён на доступный конструктор с семантическими checkbox-контролами, адаптивной сеткой и закреплённой сводкой.
- Каталог фиксирует цены за организацию на 30 дней: 103 000 ₽ по отдельности, 79 900 ₽ за полный комплект, экономия 23 100 ₽ или 22,43%.
- При восьми выбранных пакетах появляется рекомендация полного комплекта без автоматического изменения выбора.
- Описаны 72-часовой пробный период без карты, семидневный льготный период и сохранение фиксированной расчётной даты.
- Корпоративный уровень представлен как полный комплект с дополнительными организациями, сводной отчётностью, единым входом, аудитом, интеграциями, SLA, миграцией и обучением.
- Регистрация всегда создаёт бесплатную базу. Выбор маркетинговых пакетов сохраняется только как неактивирующее намерение для личного кабинета; `plan_slug` больше не передаётся.
- Обновлены главная страница, навигация, подвал, FAQ поддержки, SEO-тексты, JSON-LD, аналитические метки и рекомендации решений.
- Удалены старые pricing-компоненты, данные, дублирующие страницы и ссылки `?plan=`.
